Shader Experiment

Ik heb een shader geschreven waarmee de player dunner wordt en er een game van gemaakt. Het was een opdracht voor school. Ik heb veel geleerd over hoe je shaders moet schijven in HLSL.

                        Properties{
    _MainTex("Texture", 2D) = "white" {}
    _Color("Color", Color) = (0,0,0,0)
    _Amount("Extrusion Amount", Range(-1,1)) = 0.5
    _BumpMap("Bumpmap", 2D) = "bump" {}
    _RimColor("Rim Color", Color) = (0.26,0.19,0.16,0.0)
    _RimPower("Rim Power", Range(0.5,8.0)) = 3.0
}
