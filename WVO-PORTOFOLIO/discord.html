Hosting

Ik had de bot gehost via AutoCode. Ik kon er ook voor kiezen om het op mijn eigen pc te hosten of via een server. De voordelen via zelf hosten of via een server zijn:

  • Je kan het op je eigen pc hosten, waardoor je meer vrijheid hebt.
  • Je kan het op een server hosten en dat is iets makkelijker.

De nadelen:

  • Als je het zelf host, dan moet je computer aanstaan.
  • Als je het zelf host, dan moet je ook stabiele wifi hebben.
  • Als je het via een server host, dan heb je iets minder vrijheid.

Je kan verschillende codeer talen gebruiken zoals: JavaScript en Python. Ik koos voor JavaScript, omdat het me wel interresant leek om iets in JavaScript te maken en ook meer kennis opdoen hoe je met API's werkt.

Ik heb met behulp van wat tutorials en documentatie iets werkend gekregen.
