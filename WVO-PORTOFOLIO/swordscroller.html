Sword Scroller

Sword Scroller

We hadden een game gemaakt voor een game jam op school. We kregen een thema en we moesten 2 engelse werkwoorden toepassen in de game. De werkwoorden werden random gegenereerd.

Thema

De thema was One Button, we moesten een game maken met een knop. We hadden als werkwoorden: tear, encounter en cheat gekregen. Als knop hadden we de scrollwheel gekozen, omdat we de scrollwheel mochten gebruiken en je kan de scrollwheel indrukken.

Gameplay

Voor de gameplay hadden we als idee een 2d platformer waarmee je je zwaard kan ronddraaien en kan springen waar de zwaard naartoe richt. De eerste dag waren we gelijk begonnen aan de core gameplay en hadden we snel een werkend prototype.

Scrolling mechanic

Eerst hadden we ons gefocussed op de input van de player zodat we gelijk al wisten wat we ermee gingen doen. We hadden dus de zwaard om de player laten draaien door middel van scrollen

We hadden uiteindelijk de scrollwheel input gelocked tot een hoeveelheid graden met code zodat de player niet naar beneden kan springen.

De code kon wel wat netter door Mathf.Clamp() te gebruiken.

                        if (Input.mouseScrollDelta.y > 0 && scroll > -110)
    scroll -= scrollAmount * Time.deltaTime;
if (Input.mouseScrollDelta.y < 0 && scroll < 110)
    scroll += scrollAmount * Time.deltaTime;

Jump mechanic

Voor de jump mechanic hadden wij als de player op de grond is dat hij dan kan springen. Dat is ons gelukt door middel van de scrollwheel drukken als input te gebruiken en een overlap circle te gebruiken. Hieronder kan je de jump code zien:

if (isGrounded && Input.GetMouseButton(2))
    {
        rb.velocity = sword.up * jumpStrenght;
    }
